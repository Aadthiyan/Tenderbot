# HackerEarth Submission: TenderBot Global

## 🌟 Pitch Statement
**TenderBot Global** is an autonomous AI agent network that discovers, scores, and deeply analyzes multi-million dollar public sector tenders worldwide. We replace hundreds of manual procurement hours by leveraging Llama 3.1 to score raw RFPs against individual company capabilities. 

## 🏗️ Architecture & "Enterprise Swarms"
TenderBot is built using a secure Multi-Tenant model ideal for SMBs. However, our architecture is built to scale into **Dedicated Enterprise Agent Deployments**. 

Unlike passive chatbots or simple RAG tools, TenderBot deploys **Autonomous Agent Swarms**. For major contractors, we spin up completely isolated AI agents acting as digital employees. These agents actively navigate competitor websites to calculate market share, autonomously crawl government vendor databases to assemble sub-contractor teaming alliances, and literally navigate into complex SAM.gov applications to fill out multi-page submission forms on behalf of the client—saving thousands of human hours in actual active execution, not just text generation.

## ⚙️ Tech Stack
-   **Frontend:** Next.js, React, Tailwind CSS
-   **Backend:** FastAPI (Python), MongoDB Atlas, APScheduler 
-   **LLM Intelligence:** Fireworks.ai (Llama 3.1 70B Instruct) 
-   **Agent Web Crawling:** TinyFish
-   **Events & Monitoring:** Composio (Slack Alerts), AgentOps (Observability)
-   **Voice Interface:** ElevenLabs (Text-to-Speech Briefings)

## 🚀 Key Features
1. **Global Portal Aggregation:** Native integrations with SAM.gov, TED EU, UNGM, Find-a-Tender, AusTender, and CanadaBuys.
2. **AI Eligibility Scoring:** Llama 3.1 evaluates complex 200-page government PDFs to verify the 100+ compliance factors against user profiles.
3. **Daily Voice Briefings:** ElevenLabs synthesizes high-priority insights into an executive-level daily podcast.
4. **Real-time Slack Alerts:** Composio pipes zero-day contract amendments directly to authorized procurement channels.
